Class buRRRn.ASTUce.Assertion

  • vegas.core.CoreObject
    • buRRRn.ASTUce.Assertion

Description

A set of assert methods. Messages are only displayed when an assert fails. Assertion is used because Assert is a reserved ECMAScript futur keyword.
