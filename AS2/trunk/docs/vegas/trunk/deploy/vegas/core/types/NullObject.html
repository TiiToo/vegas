Class vegas.core.types.NullObject

  • vegas.core.CoreObject
    • vegas.core.types.NullObject

Description

Utilitarian object allowing to treat a null value as an object. A Null Object can be useful in recursive structures, typical of CompositePattern. But it is also useful in other contexts too. For example, it's commonly used in StrategyPattern (where no particular strategy is needed). Usefull for some polymorphic situation.
