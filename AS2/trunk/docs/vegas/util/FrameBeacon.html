Class vegas.util.FrameBeacon

  • vegas.util.FrameBeacon

Description

This singleton create a virtual Frame interval with a onEnterFrame event. The use can register object listeners to receive onEnterFrame event.

Example :

 import vegas.util.FrameBeacon ;
 
 var i:Number = 0 ;
 var max:Number = 25 ;
 
 var o = {} ;
 o.onEnterFrame = function () 
 {
     trace (i) ;
     i++ ;
     if (i == max) 
     {
         FrameBeacon.removeFrameListener(this) ;
     }
 }
 
 FrameBeacon.addFrameListener(o) ;
 
 this.onKeyDown = function () 
 {
     if (FrameBeacon.running)
     {
         FrameBeacon.stop() ;
     }
     else
     {
         FrameBeacon.start() ;
     }
 }
 Key.addListener(this) ;
