Class vegas.string.HTMLStringFormatter

  • vegas.string.HTMLStringFormatter

Description

This static tool class defined each of the following methods returns a copy of the string wrapped inside an HTML tag. For example, HTMLStringFormatter.bold("test") returns <b>test</b>. Compatibility with Javascript String HTML wrappers, this methods are native in the Javascript String class.

static public function anchor(str:String, name:String):String

Creates an HTML anchor that is used as a hypertext target.

 var s:String = HTMLStringFormatter.anchor("contents_anchor", "ekameleon link") ;
 trace("anchor : " + s) ; // anchor : <a name="contents_anchor">ekameleon link</a>
