getLogger

static public function getLogger(category:String, isQueue:Boolean):ILogger

Returns the logger associated with the specified category. If the category given doesn't exist a new instance of a logger will be returned and associated with that category. Categories must be at least one character in length and may not contain any blanks or any of the following characters: []~$^&\/(){}<>+=`!#%?,:;'"@

Parameters

Return

An instance of a logger object for the specified name. If the name doesn't exist, a new instance with the specified name is returned.

Throws

InvalidCategoryErrorif the category specified is malformed.
