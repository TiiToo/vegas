Description

Bit represents a bit value. This class is too an enumeration of all bit and bytes values.

Bit can be used for a different kind of formatting of a bit value. It allows to access the value as bit, kilo-bit, mega-bit, giga-bit, tera-bit, byte, kilo-byte, mega-byte, giga-byte and tera-byte.

Examples:

 
   import vegas.core.types.Bit ;
   trace(new Bit(1).toString()) ; // 1b
   trace(new Bit(1234).toString()) ; // 1.21Kb
   trace(new Bit(15002344).toString()) ; // 14.31Mb
