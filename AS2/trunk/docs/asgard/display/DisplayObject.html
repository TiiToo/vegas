Class asgard.display.DisplayObject

  • vegas.core.CoreObject
    • vegas.events.AbstractCoreEventDispatcher
      • asgard.display.DisplayObject

Description

The DisplayObject class is the base class for all objects that can be displayed in the Flash Player.
