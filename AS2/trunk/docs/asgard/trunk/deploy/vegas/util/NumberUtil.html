toFixed

static public function toFixed(n:Number, fractionDigits:Number):String

Returns a string representing the number in fixed-point notation (*ECMA-262*).

Example :

 import vegas.util.NumberUtil ;
 
 var n:Number = 12345.6789 ;
 
 trace( NumberUtil.toFixed(n) ) ;
 // Returns 12346: note rounding, no fractional part
 
 trace( NumberUtil.toFixed( n , 1) ) ;
 // Returns 12345.7: note rounding
 
 trace( NumberUtil.toFixed( n , 6 ) );
 // Returns 12345.678900 : note added zeros

Parameters

nthe number to fixed.
fractionDigitsthe number of digits to appear after the decimal point; this may be a value between 0 and 20, inclusive, and implementations may optionally support a larger range of values. If this argument is omitted, it is treated as 0.

Return

A string representation of number that does not use exponential notation and has exactly fractionDigits digits after the decimal place. The number is rounded if necessary, and the fractional part is padded with zeros if necessary so that it has the specified length. If number is greater than 1e+21, this method simply calls Number.toString() and returns a string in exponential notation.

toPrecision

static public function toPrecision(n:Number, precision:Number):String

Returns a string representing the number to a specified precision in fixed-point notation (*ECMA-262*).

Example :

 import vegas.util.NumberUtil ;
 var num:Number = 5.123456 ;
 
 trace( NumberUtil.toPrecision(num) )  ; // 5.123456
 trace( NumberUtil.toPrecision(num, 4) ) ; // 5.123
 trace( NumberUtil.toPrecision(num, 2) ) ; // 5.1
 trace( NumberUtil.toPrecision(num, 1) ) ; // 5
 trace( NumberUtil.toPrecision(1250, 2) ) ; // 1.3e+3
 trace( NumberUtil.toPrecision(1250, 5) ) ; // 1250.0

Parameters

nThe number to format.
precisionAn integer specifying the number of digits after the decimal point.

Return

A string representing a Number object in fixed-point or exponential notation rounded to precision significant digits.
