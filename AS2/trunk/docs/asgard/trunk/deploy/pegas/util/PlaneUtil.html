Class pegas.util.PlaneUtil

  • pegas.util.PlaneUtil

Description

Static tool class to manipulate and transform Plane references.

Method Index

getPlaneByObject(), getPlaneByVector()

Method Detail

getPlaneByObject

static public function getPlaneByObject(o:Object):Plane

Returns the Plane reference defines by the specified object with the properties a, b, c and d.

Example :

 import pegas.geom.Plane ;
 import pegas.util.PlaneUtil ;
 
 var p:Plane = PlaneUtil.getPlaneByObject( { a:10 , b:10 , c:100 , d:100 } ) ;
 trace(p) ; // [Plane:{10,10,100,100}]

Return

the Plane reference defines by the specified object with the properties a, b, c and d.

getPlaneByVector

static public function getPlaneByVector(v1:Vector2, v2:Vector2):Plane

Returns the Plane reference defines by the specified object with the two vectors in argument.

Example :

 import pegas.geom.Plane ;
 import pegas.geom.Vector2 ;
 import pegas.util.PlaneUtil ;
 
 var v1:Vector2 = new Vector2(10,10) ;
 var v2:Vector2 = new Vector2(100,100) ;
 var p:Plane = PlaneUtil.getPlaneByVector( v1, v2 ) ;
 trace(p) ; // [Plane:{10,10,100,100}]

Return

the Plane reference defines by the specified object with the two vectors in argument.
