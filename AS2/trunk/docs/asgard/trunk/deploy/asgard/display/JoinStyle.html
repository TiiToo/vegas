Class asgard.display.JoinStyle

  • asgard.display.JoinStyle

Description

The JointStyle class is an enumeration of constant values that specify the joint style to use in drawing lines. These constants are provided for use as values in the joints parameter of the lineStyle() method. The method supports three types of joints : 'miter', 'round', and 'bevel'.
