    41  /**
    42   * The FileReference class provides a means to upload and download files between a user's computer and a server. 
    43   * An operating-system dialog box prompts the user to select a file to upload or a location for download. 
    44   * Each FileReference object refers to a single file on the user's disk and has properties that contain information 
    45   * about the file's size, type, name, creation date, modification date, and creator type (Macintosh only).
    46   * @author eKameleon
