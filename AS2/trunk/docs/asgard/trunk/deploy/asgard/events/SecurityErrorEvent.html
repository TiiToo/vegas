Description

The SecurityErrorEvent are dispatched to report the occurrence of a security error. Security errors reported through this class are generally from asynchronous operations, such as loading data, in which security violations may not manifest immediately. Your event listener can access the object's text property to determine what operation was attempted and any URLs that were involved. There is one type of security error event: SecurityErrorEvent.SECURITY_ERROR.
