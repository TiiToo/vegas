Description

Represents a version number for a constructor, an environnement, a platform or other.

Informations :

 		1 5 432 10
 		0x  F F FFF FF
 		| |  |   |
 		| |  |   \revision (max 255)
 		| |  \build (max 4095)
 		| \minor (max 15)
 		\major (max 15)
 		
 		max version number is v15.15.4095.255

Note : value ranges are

  • 0xF - 0 -> 15
  • 0xFF - 0 -> 255
  • 0xFFF - 0 -> 4095
Max total value = 0xFFFFFFF
