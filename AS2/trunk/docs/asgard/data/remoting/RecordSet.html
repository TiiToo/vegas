getItemID

public function getItemID(index:Number)

Returns a unique ID corresponding to the record, at the specified index. The RecordSet object assigns each record a unique ID. The ID is not part of the record; it is a separate item that is associated with the record internally within the RecordSet object. Unlike a record index, its ID will not change when the RecordSet object is sorted or when records are added or deleted. When a record is deleted, its ID is retired and will never be used again in this RecordSet object.
