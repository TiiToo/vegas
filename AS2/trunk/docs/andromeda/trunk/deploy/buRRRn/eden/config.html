Field Detail

compress

static compress:Boolean
StaticProperty: compress Parameter to remove (true) or add (false) all unecessary spaces, tabs, carriages returns, lines feeds etc. to optimize (more or less) packets of datas when they are transfered. note: use "compress = false" when you want to have a better view or debug packets of datas.

copyObjectByValue

static copyObjectByValue:Boolean
StaticProperty: copyObjectByValue Parameter allowing to copy objects by value if true or by reference if false. exemple: foo = {a:1, b:2, c:3}; bar = foo; in this case with copyObjectByValue = false bar will be a reference to the foo object but if copyObjectByValue = true bar will be an exact copy of foo object

strictMode

static strictMode:Boolean
StaticProperty: strictMode Allows to define the case-sensitivy of the parsers. If true, variable names that differ only in case are considered different.

undefineable

static undefineable
StaticProperty: undefineable Value assigned to a variable when this one is not found or not authorized. note: Depending on your environment you can override it with a more suitable one for exemple on C# you could set it to null.

verbose

static verbose:Boolean
StaticProperty: verbose Parameter allowing to trace messages in the console if the environment permit it.

security

static security:Boolean
StaticProperty: security Parameter setting on (true) or off (false) the security. If true, all object path, function or constructor will be scanned at interpretation time against the authorized list (see: buRRRn.eden.config.authorized).

authorized

static authorized:Array
StaticProperty: authorized List of authorized keywords, objects path and constructors that the parser is allowed to interpret. note: you can add full path ex: "blah.foobar" and/or starting path ex: "toto.titi.*" the difference is with a full path you can only create/use/define/assign value to this exact path and with a starting path you can create/use/define/assign value to this path and its child paths attention: special values as NaN, true, false, null, undefined are always authorized

allowFunctionCall

static allowFunctionCall:Boolean
StaticProperty: allowFunctionCall Allows to execute function call is set to true, set to false it blocks any functrion call and return undefined. exemple: "titi = \"hello world\"; toto = titi.toUpperCase();" with allowFunctionCall= true toto will equal "HELLO WORLD" but with allowFunctionCall = false toto will equal undefined
