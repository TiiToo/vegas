Class asgard.events.ProgressEvent

  • vegas.core.CoreObject
    • vegas.events.BasicEvent
      • asgard.events.ProgressEvent

Description

The ProgressEvent are dispatched when a load operation has begun or a socket has received data. These events are usually generated when SWF files, images or data are loaded into Flash Player. There are two types of progress events: ProgressEvent.PROGRESS and ProgressEvent.SOCKET_DATA.
