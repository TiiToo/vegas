attachVideo

Specifies whether audio should be sent over the stream (from a Microphone object passed as source) or not (null passed as source). This method is available only to the publisher of the specified stream.

getTimeString

public function getTimeString(pattern:String):String

Returns the formatted string representation of the time position of the stream passed in argument.

Parameters

patternthe optional string pattern to format the hours, the minutes and the seconds of the passed-in time. By default the method use the static constant DEFAULT_TIME_PATTERN with the value "{0}:{1}:{2}".
  • {0} is the representation of the hours value.
  • {1} is the representation of the minutes value.
  • {2} is the representation of the seconds value.

    • Return

    the formatted string representation of the time position of the stream passed in argument.

    receiveVideo

    public function receiveVideo(value):Void

    Specifies whether incoming video will play on the specified stream, or specifies the frame rate of the video. This method is available only to clients subscribed to the specified stream, not to the stream's publisher.

    Parameters

    valueA Boolean value that specifies whether incoming video plays on the specified stream (true) or not (false). The default value is true. Or a number that specifies the frame rate per second of the incoming video. The default rate is the frame rate of the FLV file or live stream being played.
