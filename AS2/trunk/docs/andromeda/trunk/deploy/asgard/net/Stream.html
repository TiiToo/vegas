Description

The Stream class opens a one-way streaming connection between Flash Player and RTMP server through a connection made available by a NetConnection object. A Stream object is like a channel inside a NetConnection object; this channel can either publish audio and/or video data, using NetStream.publish(), or subscribe to a published stream and receive data, using NetStream.play(). You can publish or play live (real-time) data and previously recorded data.
