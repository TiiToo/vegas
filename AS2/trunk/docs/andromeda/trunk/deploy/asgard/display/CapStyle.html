Class asgard.display.CapStyle

  • asgard.display.CapStyle

Description

The CapsStyle class is an enumeration of constant values that specify the caps style to use in drawing lines. The constants are provided for use as values in the caps parameter of the lineStyle() method. You can specify the following three types of caps : NONE, ROUND or SQUARE.
