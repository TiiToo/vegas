Description

A Binary tree BinaryTree register datas in a recursive manner so that you can access it quickly by using a key.

Therefore, a BinaryTree automatically sorts data as it is inserted.

For a BinaryTree to be valid, every node has to follow two rules:

  • The data value in the left subtree must be less than the data value in the current node.
  • The data value in the right subtree must be greater than the data value in the current node.
