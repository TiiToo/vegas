Class vegas.util.MathsUtil

  • vegas.util.MathsUtil

Description

The MathUtil utility class is an all-static class with methods for working with numbers.

Method Index

ceil(), clamp(), floor(), getPercent(), round(), sign()

Method Detail

ceil

static public function ceil(n:Number, floatCount:Number):Number

Rounds and returns the ceiling of the specified number or expression. The ceiling of a number is the closest integer that is greater than or equal to the number.

Example :

 import vegas.util.MathsUtil ;
 var n = MathsUtil.ceil(4.572525153, 2) ;
 trace ("n : " + n) ; // n : 4.58
 
 var n = MathsUtil.ceil(4.572525153, -1) ;
 trace ("n : " + n) ; // n : 5

Parameters

nthe number to round.
floatCountthe count of number after the point.

Return

the ceil value of a number by a count of floating points.
