getMembersConstant
public const getMembers:Function

Returns all the public members of an object, either by key or by value.

Example :

     import core.objects.getMembers ;
     
     var o:Object = { a : 5 , b : 6 } ;
     
     trace( dump( getMembers( o ) ) ) ; // [a,b]
     trace( dump( getMembers( o , true ) ) ) ; // [5,6]

mergeConstant 
public const merge:Function

Merging enumerable properties from a specific Object to a target Object.

Example :

     import core.dump ;
     import core.objects.merge ;
     
     var target:Object = { a : 5 , b : 6 } ;
     var from:Object   = { a : 1 , b : 2 , c: 3 } ;
     
     trace( dump( merge( target , from ) ) ) ; // {a:1,b:2,c:3}
