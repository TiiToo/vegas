Classpublic class ArgumentsParser

Parse command line arguments.

Example :

     my.exe -h
     my.exe /?
     my.exe -s -T:hello myfile.txt
     my.exe myfile1.txt myfile2.txt

Note :we don't want to use events for Tamarin compatibility.
