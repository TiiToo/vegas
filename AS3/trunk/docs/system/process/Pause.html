Packagesystem.process
Classpublic dynamic class Pause
InheritancePause CoreAction Task flash.events.EventDispatcher
Implements Serializable, Startable, Stoppable

This Action object create a pause in the process.

Example : 

     import system.events.ActionEvent ;
     import system.process.Pause ;
     
     var handleEvent:Function = function( e:ActionEvent ) :void
     {
         trace( this + " " + args) ;
     }
     
     var p:Pause = new Pause( 10 ) ; // 10 seconds
     
     p.addEventListener( ActionEvent.START  , handleEvent ) ;
     p.addEventListener( ActionEvent.FINISH , handleEvent ) ;
     
     p.run() ;
