ceil()method
public static function ceil(n:Number, floatCount:Number):Number

Rounds and returns the ceiling of the specified number or expression. The ceiling of a number is the closest integer that is greater than or equal to the number.

Example :

         import system.numeric.Mathematics ;
         
         var n = Mathematics.ceil(4.572525153, 2) ;
         trace ("n : " + n) ; // n : 4.58
         
         var n = Mathematics.ceil(4.572525153, -1) ;
         trace ("n : " + n) ; // n : 5
Parameters
n:Number — the number to round.
 
floatCount:Number — the count of number after the point.

Returns
Number — the ceil value of a number by a count of floating points.
clamp()method 
public static function clamp(value:Number, min:Number, max:Number):Number

Bounds a numeric value between 2 numbers.

Example :

         import system.numeric.Mathematics ;
         
         var n = Mathematics.clamp(4, 5, 10) ;
         trace ("n : " + n) ; // 5
         
         var n = Mathematics.clamp(12, 5, 10) ;
         trace ("n : " + n) ; // 10
         
         var n = Mathematics.clamp(6, 5, 10) ;
         trace ("n : " + n) ; // 5
         
         var n = Mathematics.clamp(NaN, 5, 10) ;
         trace ("n : " + n) ; // NaN
Parameters
value:Number — the value to clamp.
 
min:Number — the min value of the range.
 
max:Number — the max value of the range.

Returns
Number — a bound numeric value between 2 numbers.
