Classpublic class DateEvent
InheritanceDateEvent BasicEvent flash.events.Event

The DateEvent to dispatch an event who contains a specific Date context.

Example : 

     import system.events.DateEvent ;
     var e:DateEvent = new DateEvent("change", new Date()) ;
     trace( e.date ) ; // true
