Interfacepublic interface InteractiveConsole extends Console, Readable, Writeable
Implementors VirtualConsole

An InteractiveConsole can interact with the user and the system using standard streams : standard input (stdin), standard output (stdout) and standard error (stderr).

See : http://en.wikipedia.org/wiki/Standard_streams.
