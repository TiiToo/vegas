getClassByName()method
public static function getClassByName(name:String):Class

Returns the class reference from a string class name. The string name notation can be either "flash.system::Capabilities" or "flash.system.Capabilities" but you have ot provide the full qualified path of the class "Capabilities" alone will not work.
