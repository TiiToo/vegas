normalize()method 
public static function normalize(v:Vector2, thickness:Number = 1):void

Scales the line segment between (0,0) and the current point to a set length.

Example :

         import pegas.geom.Vector2 ;
         import pegas.util.Vector2Util ;
         
         var v:Vector2 = new Vector2(0,5) ;
         
         Vector2Util.normalize(v) ;
         
         trace(v) ; // [Point:{0,1}]
Parameters
v:Vector2 — The scaling value. For example, if the current point is (0,5), and you normalize it to 1, the point returned is at (0,1).
 
thickness:Number (default = 1)

Throws
— if a zero-length vector or a illegal NaN value is calculate in this method.
