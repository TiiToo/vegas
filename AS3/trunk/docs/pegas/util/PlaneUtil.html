public static function getPlaneByObject(o:Object):Plane

Returns the Plane reference defines by the specified object with the properties a, b, c and d.

Example :

import pegas.geom.Plane ; import pegas.util.PlaneUtil ; var p:Plane = PlaneUtil.getPlaneByObject( { a:10 , b:10 , c:100 , d:100 } ) ; trace(p) ; // [Plane:{10,10,100,100}] Parameters
