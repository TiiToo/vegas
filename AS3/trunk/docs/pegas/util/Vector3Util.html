project()method 
public static function project(v:Vector3, perspective:Number):void

Performs a perspective projection on a 3d point. It converts (x, y, z) coordinates to a 2d location (x, y) on the screen.

Parameters
v:Vector3 — the Vector3 reference.
 
perspective:Number — The perspective ratio. If no value is specified, it is calculated automatically by calling the getPerspective() method.

Example :

var v:Vector3 = new Vector3(50,20,40) ; Vector3Util.project(v); trace(v) ;
