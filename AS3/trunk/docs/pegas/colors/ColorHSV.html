  for (var prop:String in o)

hsv2rgb()method
public static function hsv2rgb(oColor:Object, s:Number, v:Number):Object

Converts a HSV color in a RGB color.

Example :

    import pegas.colors.ColorHSV;
   var oRGB : Object = {r:250,g:0,b:0};
   var o2 : Object = ColorHSV.hsv2rgb (oRGB, 50, 100); // desaturate 50%
   for (var prop:String in o2) 
   {
       trace( prop + " : " + o2[prop] );
   }
Parameters
oColor:Object — an Object with r,g,b properties ( range from 0 - 255 ), this is the Hue.
 
s:Number — a Number represent the Saturation value ( range from 0 - 100 ).
 
v:Number — a Number represent the Brightness ( Value ) ( range from 0 - 100 ).

Returns
Object — an Object with r,g,b properties.
rgb2hsv()method 
public static function rgb2hsv(r:Number, g:Number, b:Number):Object

Converts a RGB color in a HSV color.

Example :

   import pegas.colors.ColorHSV;
   var o:Object = ColorHSV.rgb2hsv(255, 0, 0);
   for (var i in o) 
    {
     if (typeof (o[i]) == "object") 
    {
     trace(i+" : ");
     for (var j in o[i]) 
     {
      trace(" "+j+" : "+o[i][j]);
      }
     }
    else 
    {
     trace(i+" : "+o[i]);
    }
   }
Parameters
r:Number — a Number represent red value ( range from 0 - 255 ).
 
g:Number — a Number represent red value ( range from 0 - 255 ).
 
b:Number — a Number represent red value ( range from 0 - 255 ).

Returns
Object — an Object with h ( Hue ), s ( Saturation ), v ( Value / Brightness ) properties. h is an Object with r,g,b properties.
