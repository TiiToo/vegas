getRGB2()method 
public function getRGB2():Object

Returns the R+G+B values currently in use by the Color object as individual red, green, and blue values.

Example :

var my_color:Color = new Color(my_mc); my_color.setRGB2(255, 0, 255); var rgb:Object = my_color.getRGB2(); trace (rgb.r); trace (rgb.g); trace (rgb.b);
