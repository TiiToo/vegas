CMY2RGB()method 
public static function CMY2RGB(cmy:CMY):RGB

Returns the RGB representation of the specified CMY object.

Example :

         import pegas.colors.Colors ;
         import pegas.colors.CMY ;
         import pegas.colors.RGB ;
         
         var cmy:CMY = new CMY( 0 , 1 , 1 ) ;
         var rgb:RGB = Colors.CMY2RGB( cmy ) ;
         
         trace( rgb ) ; // [RGB r:255 g:0 b:0 hex:0xFF0000]

RGBA2Number()method 
public static function RGBA2Number(r:uint = 0, g:uint = 0, b:uint = 0, a:Number = 1):uint

Converts the red, green, blue, alpha values in a decimal number color value (base 10).

         trace( rgba ) ; // 0x99AAAAAA
Parameters

g:uint (default = 0) — The green component (between 0 and 0xFF)
 
b:uint (default = 0) — The blue component (between 0 and 0xFF)
 
a:Number (default = 1) — The alpha component (between 0 and 1)

Returns
