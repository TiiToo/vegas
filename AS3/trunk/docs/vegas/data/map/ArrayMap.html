Packagevegas.data.map
Classpublic class ArrayMap
InheritanceArrayMap CoreObject
ImplementsMap
SubclassesSortedArrayMap

ArrayMap is an Map implementation based on two arrays to defines the collections of the keys and the values. This implementation sort all elements in the map with the natural order of the keys and values arrays.
