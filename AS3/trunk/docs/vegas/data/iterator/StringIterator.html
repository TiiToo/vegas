Packagevegas.data.iterator
Classpublic class StringIterator
InheritanceStringIterator CoreObject
ImplementsIterator

Converts a string to an iterator.

Example :

     import  vegas.data.iterator.Iterator ;
     import  vegas.data.iterator.StringIterator ;
     
     var s = "Hello world" ;
     
     var it:Iterator = new StringIterator(s) ;
     it.seek(1) ;
     while(it.hasNext())
     {
         var char:String = it.next() ;
         trace (it.key() + ' : ' + char) ;
     }
     trace (s) ;
