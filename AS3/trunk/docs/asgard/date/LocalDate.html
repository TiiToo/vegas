Packageasgard.date
Classpublic class LocalDate

This static enumeration class register all string constants to defined a date.

Example :

  import asgard.date.LocalDate ;
  trace( "days   : " + LocalDate.getDays() ) ;
  trace( "months : " + LocalDate.getMonths() ) ;
