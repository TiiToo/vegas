Classpublic class Pause
InheritancePause Action SimpleAction AbstractCoreEventDispatcher CoreObject
SubclassesMessage

This IAction object create a pause in the process.

Example :

     var handleEvent:Function = function( ...args:Array ) :void
     {
         trace( this + " " + args) ;
     }
     var p:Pause = new Pause(10, true) ;
     p.duration = 2 ;
     p.addEventListener( ActionEvent.START  , handleEvent ) ;
     p.addEventListener( ActionEvent.FINISH , handleEvent ) ;
     p.run() ;
