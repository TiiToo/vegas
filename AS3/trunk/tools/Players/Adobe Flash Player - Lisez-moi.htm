Utilisation des versions de déboguage et de distribution d'Adobe Flash Player

Adobe® Flash® CS3 Professional est doté d'un débogueur intégré. Pour bénéficier de cette fonction, vous devez utiliser des versions spéciales d'Adobe® Flash® Player (versions de déboguage). Les versions de déboguage de Flash Player sont identiques aux versions de distribution, si ce n’est qu'elles comportent du code supplémentaire leur permettant de communiquer avec Flash CS3 Professional.

Pour simplifier les opérations de développement et de déboguage, le programme d'installation de Flash CS3 Professional installe par défaut les versions de déboguage du contrôle ActiveX ou du plug-in dans le répertoire du navigateur. Notez toutefois que les versions de Flash Player disponibles en téléchargement sur le site Web d'Adobe sont les versions de distribution : les versions de déboguage sont uniquement disponibles avec Flash CS3 Professional.

C’est pourquoi Flash CS3 Professional comprend des programmes d'installation et des lecteurs, tous signés numériquement, pour le plug-in, le contrôle ActiveX et le lecteur autonome (projecteur). La signature numérique certifie que les fichiers sont authentiques et n'ont pas subi de modifications illicites.

Fichiers inclus

Après l'installation, les répertoires Players sous Windows (Adobe/Adobe Flash CS3/Players) et sur Mac (Adobe Flash CS3/Players) ont la structure suivante.

Ces répertoires contiennent les fichiers qui seront utilisés par Flash pour créer des projecteurs ou afficher du contenu autonome. Les versions d'Adobe Flash Player placées dans ce dossier par le programme d’installation de Flash CS3 Professional sont les versions de déboguage.

  Adobe Flash Player - Lisez-moi.htm - Ce fichier.
  FlashPlayer.exe, Flash Player.app- Ce lecteur autonome sera utilisé pour afficher du contenu localement, par exemple si l’utilisateur double-clique sur un fichier SWF.
  Debug/ - Ce répertoire contient les copies de sauvegarde des versions de déboguage d'Adobe Flash Player.

 

 

Install Flash Player 9 ActiveX.msi - Programme d’installation de la version de déboguage du contrôle ActiveX d'Adobe Flash Player, utilisé par Internet Explorer sous Windows.

 

 

Install Flash Player 9 Plugin.msi - Programme d’installation de la version de déboguage du plug-in d'Adobe Flash Player, utilisé par les navigateurs qui appellent le plug-in.

 

 

Install Flash Player 9 UB.dmg - Programme d’installation de la version de déboguage d'Adobe Flash Player sous Macintosh.

 

 

FlashPlayer.exe (Windows), FlashPlayer.app (Macintosh) - Ce fichier est la version de déboguage du lecteur autonome, qui permet de visionner du contenu Flash localement.

  Release/ - Ce répertoire contient les copies de sauvegarde des versions de distribution d'Adobe Flash Player.

 

 

Install Flash Player 9 ActiveX.msi - Programme d’installation de la version de distribution du contrôle ActiveX d'Adobe Flash Player, utilisé par Internet Explorer sous Windows.

 

 

Install Flash Player 9 Plugin.msi - Programme d’installation de la version de distribution du plug-in d'Adobe Flash Player, utilisé par les navigateurs qui appellent le plug-in.

 

 

Install Flash Player 9 UB.dmg - Programme d’installation de la version de distribution d'Adobe Flash Player sous Macintosh.

 

 

FlashPlayer.exe (Windows), FlashPlayer.app (Macintosh) - Ce fichier est la version de distribution du lecteur autonome, qui permet de visionner du contenu Flash localement.




Passage de la version de déboguage à la version de distribution (et vice-versa) du lecteur

Pour passer de la version de déboguage à la version de distribution (et vice-versa) du lecteur, il est nécessaire de placer les fichiers du lecteur autonome appropriés dans le répertoire Players. Vous pouvez remplacer les fichiers du lecteur Flash Player autonome du dossier parent aussi souvent que nécessaire. Considérez les dossiers « Debug » et « Release » comme des dossiers source et utilisez leur dossier parent comme dossier de travail de votre lecteur Flash Player autonome.

 

 

 

Pour utiliser la version de déboguage du lecteur autonome ou créer des projecteurs avec déboguage à l’aide de Flash CS3 Professional

 

Copiez les fichiers du dossier « Debug » dans le dossier « Players ». Veillez à bien les copier et non pas à les déplacer afin d’éviter de supprimer par inadvertance vos copies d’archive.

   

 

Pour utiliser la version de déboguage du lecteur dans votre navigateur

 

Pour utiliser les versions de déboguage du contrôle ActiveX ou du plug-in, supprimez la version actuelle du contrôle ActiveX ou du plug-in et exécutez le programme d'installation du contrôle ActiveX ou du module externe approprié à partir du dossier « Debug ». *
 

 

Pour utiliser la version de distribution du lecteur autonome ou créer des projecteurs en version de distribution à l’aide de Flash CS3 Professional

 

Copiez les fichiers du lecteur autonome Flash Player du dossier « Release » dans le dossier « Players ». Veillez à bien les copier et non pas à les déplacer afin d’éviter de supprimer par inadvertance vos copies d’archive.

   

 

Pour utiliser la version de distribution du lecteur dans votre navigateur

  Pour utiliser les versions de distribution du contrôle ActiveX ou du plug-in, supprimez la version actuelle du contrôle ActiveX ou du plug-in et exécutez le programme d'installation du contrôle ActiveX ou du module externe approprié à partir du dossier « Release ». *

 

* Remarque : Si vous exécutez le programme d'installation d'une nouvelle version de Flash Player sans supprimer d'abord le module externe ou le contrôle ActiveX installé, le processus d'installation ne sera pas lancé car les versions de déboguage et de distribution du lecteur Flash Player ont le même numéro de version. Vous devez effacer physiquement le plug-in ou le contrôle ActiveX avant de lancer le programme d'installation. Reportez-vous à la section Désinstallation d'Adobe Flash Player pour toutes instructions.




Installation d'Adobe Flash Player 9

Pour installer Adobe Flash Player 9 pour Internet Explorer ou AOL (contrôle ActiveX) sous Windows :

 

1.

Fermez le navigateur avant d'installer le nouveau contrôle ActiveX.

2.

Supprimez les versions antérieures du contrôle ActiveX. Reportez-vous à la section Désinstallation d'Adobe Flash Player pour obtenir des informations détaillées concernant la suppression des versions actuelles et antérieures du contrôle ActiveX de Flash Player.

 

3.

Double-cliquez sur le fichier « Install Flash Player 9 ActiveX.msi » pour lancer l'installation.

 

4.

Ouvrez le navigateur. Flash Player 9 doit être installé et fonctionner.


Pour installer Adobe Flash Player 9 pour Firefox, Mozilla, Netscape, CompuServe ou Opera sous Windows :

 

1.

 Fermez le navigateur avant d'installer le nouveau module externe.

 

2.

 Supprimez les versions antérieures de Flash Player. Reportez-vous à la section Désinstallation d'Adobe Flash Player pour obtenir des informations détaillées sur la suppression des versions actuelles et antérieures du plug-in de Flash Player.

 

3.

 Double-cliquez sur le fichier « Install Flash Player 9 Plugin.msi » pour lancer l'installation.

 

4.

 Ouvrez le navigateur. Flash Player 9 doit être installé et fonctionner. Pour vérifier le bon fonctionnement du module dans Netscape, choisissez Aide > A propos des modules externes dans le navigateur.


Pour installer Adobe Flash Player 9 pour Safari, Firefox, Mozilla, Netscape ou Opera sous Macintosh :

 

1.

 Fermez le navigateur avant d'installer le nouveau module externe.

2.

 Supprimez les versions antérieures de Flash Player. Reportez-vous à la section Désinstallation d'Adobe Flash Player pour obtenir des informations détaillées sur la suppression des versions actuelles et antérieures du plug-in de Flash Player.

 

3.

 Double-cliquez sur le fichier « Install Flash Player 9 UB » pour lancer l'installation.

 

4.

 Ouvrez le navigateur. Flash Player 9 doit être installé et fonctionner. Pour vérifier le bon fonctionnement du module dans Netscape, choisissez Aide > A propos des modules externes dans le navigateur.



Désinstallation d'Adobe Flash Player

Les instructions de suppression du module externe et/ou du contrôle ActiveX sont publiées sur le site Web du support de Flash Player : http://www.adobe.com/go/flashplayer_uninstall_fr.
