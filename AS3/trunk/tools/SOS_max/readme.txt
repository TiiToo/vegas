## The SOS console ## 

you can use the SOS console to create external logs in your application.

## INSTALL ##

1 - unzip the console ( SOSmax.zip in Windows or SOSmax-osx.dmg in Mac OSX)

2 - Launch the console

3 - Test the example in the AS3/trunk/bin/test/vegas/logging/SOSTarget.fla (you can use the deploy/SOSTarget.swf to test the output in the console within Flash)
